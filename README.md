### Рубежный контроль №1 по технологиям машинного обучения

### Вывод:

В ходе работы был использован датасет load_wine, содержащий числовые признаки.

Создан искусственный категориальный признак alcohol_category на основе признака alcohol.

Были добавлены пропуски в признак magnesium и выполнена их обработка 
путем замены на среднее значение.

Построен boxplot для анализа распределения признака alcohol.

Для масштабирования применен StandardScaler.

Для кодирования категориального признака использованы:
- Label Encoding (простой, но вводит порядок)
- One-Hot Encoding (более корректный, без порядка)

Таким образом, One-Hot Encoding предпочтительнее для категориальных данных.
